Mixturette
==========

Frontend development with LESS, Bower and Latte.

Before first run
----------------

Assuming you have Node.js and NPM.

**Install globally Bower**

´$ npm install -g bower´

**Update Bower components**

`$ bower update`

Update Node.js packages

`$ cd runner`

`$ npm update`

Run on windows
--------------

`$ run.bat`

Run in shell
------------

`$ node runner/index.js`

Enjoy `http://localhost:3332`
